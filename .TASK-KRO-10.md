# Task: KRO-10 - Implement Import System (Upload XLS, Read, Preview Data)

## Contexto
Implementar un sistema genérico de importación de archivos XLS/CSV que permita:
- Cargar archivos
- Parsear y mostrar preview de datos
- Mantener datos en memoria (no persistir en BD aún)
- Ser reutilizable para futuras importaciones (Properties, Users, etc.)
- Política de retención: eliminar archivos automáticamente después de 7 días

## Flujo de trabajo:
1. Usuario sube archivo XLS/CSV → XLS Upload Component
2. Componente llama a Server Action para guardar en Supabase Storage
3. Server Action retorna fileId y URL
4. Componente llama a Service para parsear el archivo
5. Service usa xlsx library para parsear y retorna ImportData
6. Componente muestra preview con Data Table
7. Usuario puede editar celdas inline y confirmar/cancelar
8. Al confirmar, se envían los datos al padre vía callback
