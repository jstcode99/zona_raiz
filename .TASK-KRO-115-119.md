### KRO-115: Forms - yupResolver y useEffect reset errors (Prioridad 1)
**Archivos:**
- `features/listing/listing-form.tsx`
- `features/properties/property-form.tsx`
- `features/real-states/real-estate-form.tsx`
- `features/profile/profile-form.tsx`
- `features/auth/sign-up-form.tsx`

**Correcciones:**
1. Eliminar `as any` de yupResolver
2. Agregar `mode: "onBlur"` en useForm
3. Agregar useEffect para resetear errores de mutation

**Patrón a seguir:**
```typescript
const form = useForm<FormInput>({
  resolver: yupResolver(schema),
  defaultValues: defaultValues,
  mode: "onBlur",
})

// Agregar useEffect:
useEffect(() => {
  const subscription = form.watch(() => {
    if (mutation.isError) mutation.reset()
  })
  return () => subscription.unsubscribe()
}, [form, mutation])
```
